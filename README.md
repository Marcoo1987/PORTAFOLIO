## ✨ Características Principales

- **Diseño Premium & Inmersivo**: Interfaz moderna con animaciones fluidas y efectos de cristal (Glassmorphism).
- **Experiencia 3D**: Integración de fondos dinámicos con **Three.js**.
- **Animaciones Cinematográficas**: Uso extensivo de **GSAP** para transiciones y efectos visuales.
- **Scroll Suave**: Implementación de **Lenis** para una navegación pulida.
- **Interactividad**: Efectos de inclinación con **Vanilla Tilt** y un cursor personalizado dinámico.
- **Totalmente Responsivo**: Optimizado para dispositivos móviles, tablets y escritorio.

---

## 🛠️ Tecnologías Utilizadas

- **Core**: HTML5 moderno, CSS3 (Vanilla) y JavaScript (ES6+).
- **Herramientas de Build**: [Vite](https://vitejs.dev/) para un desarrollo ultrarrápido.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (configurado para optimización).
- **Librerías de Animación**:
  - [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
  - [Three.js](https://threejs.org/) (Gráficos 3D en el navegador)
  - [Lenis](https://github.com/darkroomengineering/lenis) (Smooth Scroll)
  - [Vanilla Tilt](https://micku7zu.github.io/vanilla-tilt.js/)

## ⚙️ Instalación y Activación

Para ejecutar este proyecto localmente, sigue estos pasos:

### 1. Requisitos Previos

Asegúrate de tener instalado [Node.js](https://nodejs.org/) (versión 18 o superior recomendada) y `npm`.

### 2. Clonar el repositorio

```bash
git clone https://github.com/Marcoo1987/PORTAFOLIO.git
cd PORTAFOLIO
```

### 3. Instalar dependencias

```bash
npm install
```

---

## 🚀 Uso y Desarrollo

### Iniciar el servidor de desarrollo

Ejecuta el siguiente comando para ver los cambios en tiempo real:

```bash
npm run dev
```
El proyecto estará disponible en `http://localhost:5173`.

### Construcción para producción

Para generar los archivos optimizados listos para despliegue:

```bash
npm run build
```
Los archivos se generarán en la carpeta `dist/`.

### Previsualizar la construcción

```bash
npm run preview
```

## 📁 Estructura del Proyecto

- `index.html`: Punto de entrada principal.
- `src/`: Lógica principal del proyecto.
- `src/main.js`: Archivo de inicialización y orquestación de librerías.
- `src/style.css`: Estilos globales y configuraciones de diseño.
- `public/`: Archivos estáticos.
- `img/`: Activos visuales del portafolio.
