# Portafolio Personal

Este es un portafolio desarrollado como parte del Bootcamp de Python 2024.

## Estructura del Proyecto

- **MiPortafolio2.html**: Archivo principal HTML del portafolio.
- **Styles.css**: Hojas de estilo para el diseño visual.
- **script.js**: Lógica y funcionalidades interactivas en JavaScript.
- **img/**: Carpeta para imágenes utilizadas en el portafolio.
- **Link Portafolio.txt**: Archivo de texto con enlaces relevantes.

## Cómo visualizar el portafolio

1. Abre el archivo `MiPortafolio2.html` en tu navegador web preferido.
2. Asegúrate de que los archivos `Styles.css`, `script.js` y la carpeta `img/` estén en la misma carpeta que el archivo HTML.
